android studio

an individual project built in java using android studio, developed alongside my ui/ux module to practice the full product design cycle from prototype to implementation.

features
  • playback controls: play, pause, next, previous, repeat, and shuffle
  • register and delete user accounts
  • firebase authentication for secure user login

ui/ux

design process

before writing any code, i designed the app's interface in Adobe XD, creating both lo-fi wireframes to map out the user flow and hi-fi prototypes to finalise the visual design.

to validate the design, i conducted usability testing and collected user feedback using the system usability scale (SUS) survey, using the results to refine the ui before development.
